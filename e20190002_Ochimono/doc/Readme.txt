■ゲームの概要



　★テスト中
　　当ゲームは動作確認が取れていません。



　ぷよぷよをオマージュした落ちものゲームです。
　パズルのみ。ストーリーはありません。

　１人プレイ（対ＣＰＵ）
　２人プレイ（対戦）
　ネットワーク対戦（要ＵＤＰポート解放）に対応しています。

　ゲームパッド対応


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　Ochimono-NT.exe を実行して下さい。


■操作方法

　●Ｐ１

　　カーソルキー　　...　上下左右
　　Ｚ　　　　　　　...　左回転（決定）
　　Ｘ　　　　　　　...　右回転（キャンセル）
　　スペースキー　　...　ポーズ

　●Ｐ２

　　テンキー１　　　...　左
　　テンキー２　　　...　下
　　テンキー３　　　...　右
　　テンキー４　　　...　左回転（決定）
　　テンキー５　　　...　上
　　テンキー６　　　...　右回転（キャンセル）


　★キー配置は「タイトルメニュー／設定／プレイヤー１(２)のキー割り当て」から変更できます。

　★キーボードによっては同時に複数キー押下を検出できない場合があります。
　　２人プレイの場合はゲームパッドの使用を推奨します。
