■ゲームの概要

　白い箱のようなものを操作してゴール地点を目指す2Dアクションゲームです。

　文字とエフェクト以外はほとんど単色の四角形で画面を構成しているため、
　見た目はシンプルというかレトロというかシュールな感じになっています。
　（グラフィックで描画するモードもあります）

　マップを編集する機能もあるので、自分で作ったマップで遊ぶこともできます。

　全９フロア（ステージ）
　難易度は高めです。

　ゲームパッド推奨


■注意

　このソフトは、フリー画像素材・フリー音楽（効果音）素材を使用してます。
　このソフト内で使用されている画像・音楽（効果音）を、このゲームを遊ぶ以外の用途には使用しないで下さい。


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　Hako.exe を実行して下さい。


■遊び方

　●操作方法

　　左右キー        ...  左右に移動
　　下キー          ...  床の際から下に降りる
　　Ｚ              ...  ジャンプ／決定
　　Ｘ              ...  低速移動／キャンセル
　　Ａ              ...  視点移動
　　Ｓ              ...  スタート地点へ戻る
　　スペースキー    ...  ポーズ（一時停止）
　　エスケープキー  ...  直ちにゲームを終了


　　ゲームパッドに対応しています。
　　ボタン設定は「設定 / ゲームパッドのボタン設定」から行ってください。

　　低速移動ボタンを押しながら左右ボタンを押すと、箱はゆっくり移動します。

　　ポーズメニューの「スタート地点へ戻る」を選択すれば、直ちにスタート地点からやり直すことができます。
　　同名のボタンはこれと同じ機能です。

　　視点移動ボタンを押しながら上下左右ボタンを押すと、視界を調整できます。
　　視点移動ボタンだけ押して離すとデフォルトの視界に戻ります。


　　▲ゲームパッドについて

　　　ゲームパッドはゲームを起動する前にＰＣに接続し認識させて下さい。
　　　プレイ中に接続しても、たぶん正しく認識されません。

　　　複数のゲームパッドを接続している場合、
　　　ゲームを開始してから最初にボタンを押したゲームパッドが自動的に選択されます。


　●操作のコツ

　　マリオのＢダッシュのように１キャラ分の隙間は走り抜けられます。
　　２キャラ分の隙間も走り抜けられます。

　　床の端や１キャラ分の隙間から下に落ちたいとき、下ボタンを押すと落ちやすくなります。

　　各キャラの座標は内部的に 0.001 ドットで制御しています。
　　見た目は同じでも当たるか当たらないかの違いがあるかもしれません。

　　速度の速い敵は、壁にぶつかった時けっこうめり込みます。
　　当たらないだろうと思ったところでも当たるかもしれません。
　　１キャラ分以上めり込むことはないはずなので、壁の向こうにいれば大丈夫。多分。。


　●マップ上のオブジェクト

　　白い箱 ... 主人公です。

　　黄色っぽい箱 ... 壁です。

　　赤く光る箱 ... 即死です。

　　水色に光る箱 ... ゴール地点です。

　　動く青い箱 ... 敵です。触ると即死です。

　　跳ねる赤い箱 ... 敵です。触ると即死です。

　　廻る緑の箱 ... 敵です。触ると即死です。


　●タイムアタック

　　全フロア制覇すると TIME ATTACK で遊べるようになります。

　　全フロア制覇とかやってられんという方は「設定 / ロックされている機能の開放」 を実行して下さい。
　　（全フロア制覇したことにして TIME ATTACK, EDIT MODE を解放します）


　●マップの編集

　　全フロア制覇すると EDIT MODE からマップの編集ができるようになります。

　　全フロア制覇とかやってられんという方は「設定 / ロックされている機能の開放」 を実行して下さい。
　　（全フロア制覇したことにして TIME ATTACK, EDIT MODE を解放します）


　●補足・ヒント

　　フロア７に散りばめられた６桁の数字は毎週木曜日午前０時（協定世界時）に全て入れ替わります。


■リプレイ

　リプレイを保存＆再生できます。
　リプレイデータは Replay フォルダにファイルとして出力するので、単独で持ち運びできます。
　リプレイデータの再生には、このゲームプログラムと当該マップデータが必要ですのでご注意下さい。

　ミスしたときのリプレイも保存できるようになりました。
　再スタート後、キーボードのＲキーを押すと直前にミスしたリプレイを保存します。

　リプレイの保存を有効にするには「設定 / リプレイの保存」を「有効」にして下さい。
　（各フロアクリア後に保存するか確認するようになります）

　●制限

　　１フロアのプレイ時間が１時間を超えるとリプレイを保存できません。
　　この場合、クリアしても保存の確認画面をスキップします。
　　（ここで言うプレイ時間とは、フロアのスタート又は最後にミスしてからの経過時間を指します）

　　作成したリプレイデータは、同じバージョンでのみ再生可能です。


■タイムアタック

　ランクは以下のようになっています。（最高ランクはＳです）

　　Ｓ > Ａ > Ｂ > Ｃ > Ｄ > Ｅ > Ｆ > Ｇ > Ｈ

　Ｓランクを獲るには、クリアタイムが「Ｓランクタイム」以内でなければなりません。
　Ｓランクタイムは以下のとおりです。

　　FLOOR 1      00:07.000
　　FLOOR 2      00:24.000
　　FLOOR 3      00:27.000
　　FLOOR 4      00:25.000
　　FLOOR 5      00:48.000
　　FLOOR 6      00:44.000
　　FLOOR 7      00:55.000
　　FLOOR 8      00:59.000
　　FLOOR 9      00:24.000
　　----------------------
　　TOTAL        05:13.000

　ランクは以下のように決定されます。

　　クリアタイムがＳランクタイム以内               ... Ｓランク
　　クリアタイムがＳランクタイムの１１０％以内     ... Ａランク
　　クリアタイムがＳランクタイムの１３０％以内     ... Ｂランク
　　クリアタイムがＳランクタイムの１７０％以内     ... Ｃランク
　　クリアタイムがＳランクタイムの２５０％以内     ... Ｄランク
　　クリアタイムがＳランクタイムの４１０％以内     ... Ｅランク
　　クリアタイムがＳランクタイムの７３０％以内     ... Ｆランク
　　クリアタイムがＳランクタイムの１３７０％以内   ... Ｇランク
　　それ以外                                       ... Ｈランク


■マップ編集モード

　「EDIT MODE / マップ編集モード」からゲームを開始するとマップの編集（自作フロアの作成）が可能になり、
　通常のプレイと同じように始まりますが、ポーズメニューの「マップを編集する」が使えるようになります。
　「マップを編集する」を選択することによりマップ編集を開始しますが、
　操作が面倒ならキーボードのＥキーを押しても同じ効果があるのでお使い下さい。
　マップ編集モードのときはエンディングをスキップします。リプレイも保存できません。

　●マップ編集時の操作方法

　　マップ編集時は主にマウスを使用します。

　　左クリック ... マウスカーソルの位置に選択しているセルを配置
　　右クリック ... マウスカーソルの位置のセルを空にする。
　　シフトキーを押しながら左ドラッグ ... 画面の移動

　　Ｅキー押下 ... マップ編集終了
　　Ｗキー押下 ... マウスカーソルの位置をワープ先として記憶

　●編集ウィンドウの説明

　　MENU POS ... ウィンドウの位置を左端または右端に移動
　　SELECT   ... 選択されているセルを表示
　　X        ... 編集終了

　　選択可能なセル

　　　WL ... 壁
　　　DT ... 即死する壁
　　　ST ... スタート地点
　　　GL ... ゴール地点

　　　★スタート地点は必ず１つだけ配置します。

　　　青いセル

　　　　6 ... 青い箱、スタート時の進行方向は右
　　　　3 ... 青い箱、スタート時の進行方向は右下

　●マップのサイズの変更

　　ゲームからマップのサイズを変更することはできません。
　　マップデータファイルを直接編集（ペイント等を使用してリサイズ）して下さい。

　　なお、マップが大きいほどロード時間や処理は長く重くなり、
　　敵の数も多くなるほど処理が重くなります。
　　巨大なマップを作成する場合はご注意下さい。


　●ワープ機能

　　編集中にミスする度にスタート地点へ戻されるのが煩わしいときは、ワープをお使い下さい。
　　マップ編集中に任意のセルにカーソルを合わせ、キーボードのＷキーを押してその位置を記憶、
　　その後プレイ中に同じくＷキーを押すと、最後に記憶した場所へ瞬時に移動します。
　　そのフロアをクリアするかプレイ中止すると位置はクリアされます。


■マップデータファイル

　マップデータは Map フォルダの配下に .bmp 形式のファイルとして置いてあります。
　なので Windows 付属のペイント等で直接編集できます。
　（マップデータファイルを直接編集する時はゲームを終了して下さい）

　ネタバレを防ぐため最初はダミーデータを置いています。
　マップ編集モードが解放されるタイミングでオリジナルのマップデータがコピーされます。
